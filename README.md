Include the file `s3-media-upload-and-serve.php` directly into your theme or plugins.


## Replace the DEFINED CONTSTANTS.

Define into file `s3-media-upload-and-serve.php` the rows :

```


S3_BUCKET_NAME => With your bucket name

S3_KEY => with your IAM Key

S3_SECRET => with your IAM Secret

S3_YOUR_WEBSITE_URL => Your website url (full with HTTP or HTTPS).

S3_CL_OR_CN_NAME => Your cloudflare or cname or s3 direct link (full with HTTP or HTTPS).


```


Enjoy.
